## Що реалізовано

У консольному застосунку C# реалізовано два завдання:

1. Обробка файлу `textPD23.txt` за допомогою делегата `TextOperation`:
   - переведення рядків у верхній регістр;
   - підрахунок кількості символів у кожному рядку;
   - підрахунок кількості слів у кожному рядку.

   Результати всіх трьох операцій записуються у файл `resultPD23.txt`.

2. Логування повідомлень користувача:
   - клас `MessagePublisher` має подію відправки повідомлення;
   - клас `FileLogger` підписується на цю подію;
   - 4 повідомлення, введені з консолі, записуються у файл `logPD23.txt` у форматі `[час] повідомлення`.

Програма автоматично створює потрібні файли, якщо їх немає. Якщо `textPD23.txt` відсутній або порожній, він заповнюється тестовими рядками.

## Як запустити

1. Відкрити термінал у папці проєкту.
2. Виконати команду:

```bash
dotnet run
```

3. Після запуску ввести 4 повідомлення у консоль.

## Файли проєкту

- `Program.cs` - основний код програми.
- `ModularWork1_PD23.csproj` - файл проєкту C#.
- `textPD23.txt` - вхідний текстовий файл.
- `resultPD23.txt` - файл з результатами обробки.
- `logPD23.txt` - файл логування повідомлень.
